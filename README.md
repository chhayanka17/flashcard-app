# FlashCraft — AI Flashcard Studio 🧠⚡

An AI-powered Notes-to-Flashcards web application that transforms raw text into interactive study flashcards using Natural Language Processing.

## 🌐 Live Demo
👉 [flashcraftstudio.netlify.app](https://flashcraftstudio.netlify.app)

## 🚀 Features
- Paste or upload notes → AI generates flashcards instantly
- Spaced repetition study mode with flip animation
- Progress tracking (Got It / Need Review)
- Session timer and completion summary
- Responsive design for desktop and mobile
